# Data Folder

This folder is used to organise the datasets referenced in the analysis.

The project works with three CSV files containing event records, device information and manufacturer details. These files were used during the development of the notebook and script and are expected to be placed in this directory when running the analysis locally.

File structure expected by the code:

- events-1681209680.csv  
- devices-1681209661.csv  
- manufacturers-1681209657.csv  

The filenames above match those used in the Python script and notebook so the data can be loaded without modifying the paths.
